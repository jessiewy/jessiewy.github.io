*args 和 **kwargs 主要用于函数的定义，可以将不确定数量的参数传递给一个函数。

## *args的用法

*args是用来发送一个非键值对的可变数量的参数列表给一个函数。



例子1：

```
def test_var_args(f_arg, *args):

    print  "first normal arg:", f_arg

    for arg in args:

        print "another arg through *argv:", arg

test_var_args(‘python’, 'IT','IS','GOOD')
```
